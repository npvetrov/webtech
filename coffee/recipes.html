Эспрессо

Ингредиенты:

  • 7-9 г свежемолотого кофе в зернах (средний помол)
  • 60-70 мл горячей воды (90-96°C)

Инструкции:

  1. Насыпьте молотый кофе в холдер кофемашины, равномерно распределив его.
  2. Утрамбуйте кофе с помощью темпера (с усилием 25-30 кг).
  3. Запустите кофемашину на 25-30 секунд. Должно получиться около 30 мл эспрессо.
  4. Наслаждайтесь!

Примечание: Время экстракции и количество воды могут варьироваться в зависимости от кофемашины и желаемого вкуса.

Капучино

Ингредиенты:

  • Эспрессо (60 мл)
  • Вспушенное молоко (120 мл)
  • Какао-порошок (по желанию)

Инструкции:

  1. Приготовьте эспрессо.
  2. Взбейте молоко до образования плотной пены.
  3. Аккуратно вылейте взбитое молоко в чашку с эспрессо.
  4. Посыпьте какао-порошком (по желанию).
  5. Наслаждайтесь!

Примечание: Соотношение эспрессо и молока может варьироваться по вашему вкусу.

Латте

Ингредиенты:

  • Эспрессо (60 мл)
  • Горячее молоко (150-180 мл)
  • Молочная пена (немного)

Инструкции:

  1. Приготовьте эспрессо.
  2. Взбейте немного молока в пену.
  3. Добавьте горячее молоко в чашку.
  4. Аккуратно влейте эспрессо.
  5. Сверху добавьте немного молочной пены.
  6. Наслаждайтесь!

Примечание: Латте - это напиток с большим количеством молока, чем капучино.

Американо

Ингредиенты:

  • Эспрессо (60 мл)
  • Горячая вода (90-120 мл)

Инструкции:

  1. Приготовьте эспрессо.
  2. Добавьте горячую воду в чашку с эспрессо.
  3. Перемешайте.
  4. Наслаждайтесь!

Примечание: Американо - это разбавленный эспрессо.

Мокачино

Ингредиенты:

  • Эспрессо (60 мл)
  • Шоколадный сироп (20-30 мл)
  • Вспушенное молоко (120 мл)
  • Взбитые сливки (по желанию)

Инструкции:

  1. Приготовьте эспрессо.
  2. Добавьте шоколадный сироп в чашку.
  3. Влейте эспрессо.
  4. Добавьте взбитое молоко.
  5. Украсьте взбитыми сливками (по желанию).
  6. Наслаждайтесь!

Примечание: Количество шоколадного сиропа регулируйте по вашему вкусу.
